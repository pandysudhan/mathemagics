# Mathemagics

Web app for hands written digits recognition

- Backend (server): 
  - Model: Convolutional Neural Network (CNN)
  - Training dataset: MNIST Dataset
  - Web Framework: Flask
  - Hosting: Huggingface spaces with docker

- Frontend (cleint):
  - Framework: React
  - Hosting: Github pages
